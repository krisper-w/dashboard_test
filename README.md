Небольшое SPA-приложение на Vue 3, для демонстрации базовых навыков работы с современным frontend-стеком. Приложение «Dashboard» с простой авторизацией-заглушкой и страницей со списком данных взятых с https://jsonplaceholder.typicode.com. Проект также включает в себя интеграцию с WebSocket.
Проект содержит три страницы:
Login: Форма авторизации.
Dashboard: Таблица со списком участников.
Details: Информация о выбранном пользователе, получаемая по динамическому ID.

Технический стек и решения:
Core: Vue 3 (Composition API), TypeScript, Pinia (управление состоянием пользователей).
UI & Стили: Использование компонентов Vuetify. Верстка выполнена с применением SCSS и методологии БЭМ.
Компонентный подход: Реализован универсальный компонент BaseCard, использующий Props для настройки стилей и Slots (именованный заголовок и стандартное тело) для гибкого наполнения контентом.
Архитектура: Использование Provide/Inject для передачи глобальной конфигурации (API URL) по всему дереву компонентов.
WebSocket: Реализован отдельный Composable-модуль для работы с WebSocket (подключение, статус соединения, отправка и получение эхо-сообщений).
